## üö® Common Issues & Solutions

### **Issue 1: AI Dance Looks Robotic**
**Solution:**
- Try different photo (more dynamic pose)
- Select different dance style (try "Freestyle" instead of "Hip-Hop")
- Shorter duration (10s instead of 15s reduces artifacts)

### **Issue 2: CapCut Watermark Won't Remove**
**Solution:**
- Subscribe to CapCut Pro ($10/month)
- Export on desktop version (better export options)
- Alternative: Use DaVinci Resolve to crop watermark (free)

### **Issue 3: Voice Not Syncing to Beats**
**Solution:**
- Use CapCut's "Beat Detection" feature
- Manually place voice on beat markers
- Adjust tempo slightly (speed up/slow down voice)

### **Issue 4: UI Overlays Look Cheap**
**Solution:**
- Use glass morphism effect (blur + transparency)
- Add subtle animations (pulsing, fading)
- Use professional fonts (Montserrat, SF Pro, Roboto)
- Add shadows/glows for depth

## üí° Pro Tips

### **CapCut Hacks:**

1. **Multiple Exports:**
   - Generate 5-10 videos (different styles)
   - Don't settle for first result
   - Mix & match best parts

2. **Speed Control:**
   - 0.75x speed = smoother, more realistic
   - 1.25x speed = more energetic
   - Try both!

3. **Transitions:**
   - Add fade in/out (0.5s)
   - Smooth transitions between sections
   - Don't make it too flashy

4. **Color Grading:**
   - Boost saturation slightly (+10%)
   - Add slight vignette
   - Makes it more cinematic

---

### **UI Design Principles:**

1. **Less is More:**
   - Don't overcrowd with text
   - 1-2 UI elements max per scene
   - White space is your friend

2. **Readability:**
   - High contrast (white text on dark bg)
   - Large fonts (readable on phones)
   - No fancy scripts (modern sans-serif only)

3. **Consistency:**
   - Same color scheme (purple/pink gradient)
   - Same font family throughout
   - Matching animation speeds
